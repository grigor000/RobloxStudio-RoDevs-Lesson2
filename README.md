MandomServer - Controls the Mandom ability system and manages player state.

MandomClient - Handles player input for Mandom abilities.

AbilitySystem - Defines abilities available for Mandom.

MandomCore - Main controller for the Mandom entity.

MandomFX - Handles all visual and audio effects.

RecordSystem - Records and replays player states for time manipulation.
